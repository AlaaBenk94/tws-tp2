### Usage
#### Preparation
1. installer les packages npm 
```bash
npm install
```
2. build 
```bash
npm build 
```
#### Production
Demarer le serveur en mode production
```bash
npm start 
```
#### Developpement
 Lancer en mode dev
```bash
npm run dev 
```
#### Deploiement
 le deploiement en heroku se fait automatiquement des le 
 push en github.
